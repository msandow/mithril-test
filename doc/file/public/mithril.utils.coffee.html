ajax =

A helper for all types of ajax requests: m.ajax.get(), m.ajax.delete(), m.ajax.post(), m.ajax.put(), m.ajax.head(), each non-blocking unlike the m.request() method

Parameters:

  • url ( String ) — the url to send the request to
  • data ( Object ) — the post variables for POST or PUT requests, or the querystring for GET, DELETE and HEAD requests
  • callback ( Function ) — the function to call on request completion with a signature of (error, response)
